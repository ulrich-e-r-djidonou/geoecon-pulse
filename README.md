## Projects featured

| Project | Stars | Category | Description |
|---------|-------|----------|-------------|
| [caveman](https://github.com/JuliusBrussee/caveman) | 62k ⭐ | Token Optimization | 65% fewer tokens by talking like caveman |
| [graphify](https://github.com/safishamsi/graphify) | 49k ⭐ | Knowledge / RAG | Turn any codebase into a queryable knowledge graph |
| [career-ops](https://github.com/santifer/career-ops) | 46k ⭐ | Career | AI-powered job search system with 14 skill modes |
| [impeccable](https://github.com/pbakaus/impeccable) | 29k ⭐ | Design | Design language that makes AI better at UI |
| [codeburn](https://github.com/getagentseal/codeburn) | 6.7k ⭐ | Cost Observability | TUI dashboard for token cost tracking |
| [design-extract](https://github.com/Manavarya09/design-extract) | 2.7k ⭐ | Design | Extract any site's full design system in one command |
| [claude-video](https://github.com/bradautomates/claude-video) | 1.3k ⭐ | Video / Media | Give Claude the ability to watch any video |

## Stack

- **Zero build** — single `index.html`, open it in any browser
- **Google Fonts** — Inter + JetBrains Mono
- Filter by category (Token, Design, Career, RAG, Cost, Video)

## Run locally

```bash
git clone https://github.com/ulrich-e-r-djidonou/geoecon-pulse.git
cd geoecon-pulse
open index.html
# or: npx serve .
```
